# Calculator Project
## STEP 1. 클래스 없이 연산 수행하기 
가장 기본적인 자바 문법을 활용하여 계산기를 구현하는 단계입니다.

### 구현 기능

- 입력 : Scanner를 통해 양의 정수(0포함) 2개를 입력받아 변수에 저장합니다.
- 정수 입력 에외처리 : 정수가 0보다 작을 경우 "양의 정수(0을 포함)를 입력해주세요" 라는 메시지를 출력하며 다시 재입력합니다.
- 연산 기호 처리 : CharAt(0)을 사용해 사칙연산(```+```, ```-```, ```*```, ```/```)을 가져옵니다.
- 사칙연산 입력 예외처리 : 사칙연산 기호(```+```, ```-```, ```*```, ```/```)가 아닌 다른 입력값인 경우 다시 기호를 입력합니다.
- 제어문 활용 : if문과 switch문을 사용하여 계산 및 예외 처리를 수행합니다.
- 나눗셈 예외 처리 : 분모가 0일 경우 "나눗셈 연산에서 분모에 0이 입력될 수 없습니다." 라는 메시지를 출력합니다.
- 무한 반복 루프 : while문을 활용하여 사용자가 exit을 입력하기 전까지 계산을 반복합니다.

**POINT**
- Scanner의 입력 버퍼 "/n" 문제를 nextLine()를 한번 더 사용해 지우는 방벙이 있습니다.

## STEP 2. 클래스를 적용해 연산 수행하기 
기존 코드를 변경하여 유지보수가 편한 계산기를 구현하는 단계입니다.

### 추가 및 개선 기능
- Calculator 클래스 생성 : 연산하는 부분을 별도의 클래스로 분리하였습니다.
- 컬렉션 필드 생성 : 연산 결과를 출력만 하는 것이 아니라 ArrayList에 저장하였습니다.
- num1과 num2가 0포함 양수가 아닐 때 예외처리를 했습니다.
- 캡슐화 : 연산 결과 리스트를 private으로 선언하여 외부의 직접 접근을 차단했습니다.
  - Getter 메서드를 구현하여 데이터를 안전하게 조회하도록 했습니다.
- 데이터 삭제 기능 : 가장 먼저 저장된 계산 데이터를 삭제하는 removeFirstList 기능을 추가했습니다.
- 데이터 수정 기능 : 리스트에 있는 데이터를 수정할 수 있는 setResult 메서드를 구현하였습니다.
  - Setter 메서드를 구현하여 데이터를 안전하게 수정하도록 했습니다.
- 반환 : calculate 메서드가 결과값을 return하여 호출한 곳 (```App.java```)에서 활용할 수 있게 했습니다.
  - add를 사용하여 리스트에 연산 결과를 저장했습니다.

## 프로젝트 구조
```
# App.java
프로그램의 메인 실행부
사용자 입출력 제어 및 Calculator 인스턴스 관리

# Calculator.java
사칙연산 비즈니스 로직 담당
연산 결과 저장 및 삭제, 수정(ArrayList)
```

### 실행
1. 프로그램을 실행하면 정수 2개와 연산자를 입력받습니다.
2. 연산 결과가 출력되며, 현재까지의 계산 기록이 리스트로 표시됩니다.
3. 추가 계산(```yes```, 기록 삭제```remove```, 기록 수정```update```, 프로그램 종료```exit```)중 원하는 명령을 입력합니다.
